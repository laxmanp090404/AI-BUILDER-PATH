# Assignment 1 — Prompt Optimization Lab

## Question : 
### 🧪 Assignment 1 — Prompt Optimization Lab

🏆 Max 75 pts

You're developing an AI-powered **customer support assistant** handling billing-related queries for a SaaS product.

**Current (basic) prompt in use:**

`You are a helpful assistant. Answer the user's question about their billing issue.`

Responses are often too generic or incomplete.

**Your task:**

1. Analyze what's lacking in the above prompt
2. Rewrite it using best practices from Prompt Engineering Guides (CLEAR framework, specificity, constraints)
3. Enhance further using **Chain-of-Thought prompting** so the assistant reasons step-by-step — especially for scenarios like late fees, refund eligibility, or incorrect charges
4. Submit the following:
    - ✅ Refined prompt
    - ✅ CoT-enhanced prompt
    - ✅ Sample responses generated for each
    - ✅ Short explanation of which worked best and why
